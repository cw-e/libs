Installing
----------

**Python 3.10 or higher is required**

To install the library you can run the following command:

.. code:: sh

    # Linux/macOS
    python3 -m pip install -U git+https://github.com/bowlsr/libs

    # Windows
    py -3 -m pip install -U git+https://github.com/bowlsr/libs
